Guía para Plan de Trabajo (F-023) Visitas

Este formato define tus actividades concertadas. Se diligencia en diferentes momentos clave:

  • Visita 1 (Concertación): Inicio de la etapa.
  • Visita 2 (Y visita 3 Parcial): Seguimiento al 40-50% de avance.
  • Visita Final: Cierre y evaluación.
  • Extraordinaria: En caso de novedades o incumplimientos.
Actividad Competencia / Resultado Evidencias Tiempo
Desarrollar módulo de autenticación segura. Implementar soluciones de software. Código fuente, Informes. Semana 1-6
Documentar manuales de usuario. Validar requisitos del software. Manual PDF, Guías. Semana 7-8
Ejecutar plan de pruebas unitarias. Verificar calidad del software. Log de pruebas, matriz de errores. Semana 9-12
Migración de base de datos a producción. Administrar bases de datos. Script SQL, Backup. Semana 15
Descargar
  • Verbos en Infinitivo: Inicia siempre con verbos como Analizar, Diseñar, Construir, Verificar.
  • Alineación: Asegúrate que la actividad tenga relación directa con lo que estudiaste. Un desarrollador de software no debería poner "Servir cafés" o "Mensajería".
  • Firmas: Es vital que este documento tenga las 3 firmas (Tú, Jefe, Instructor) desde el primer mes.
Guía para Bitácora (F-147)

Tu diario de campo quincenal o mensual. Sé descriptivo y honesto con el tiempo reportado.

Registro quincenal (extracto):

Fecha Actividad realizada Horas Observaciones
05/04/2026 Configuración de entorno local y revisión de historias de usuario asignadas para el sprint 1. 8 Se acordó estándar de ramas en Git con el equipo.
12/04/2026 Desarrollo inicial del formulario de registro y pruebas unitarias. 8 Se identificaron mejoras en validaciones de campos obligatorios.
15/04/2026 Reunión de seguimiento (Daily) y ajuste de estilos CSS en módulo de usuarios. 4 Permiso de 4 horas en la tarde por cita médica (Soporte adjunto).
Descargar Bitácora F-147
  • Vagueza: Evita "Hacer lo que me dijo el jefe". Sé técnico: "Migración de base de datos SQL".
  • Periodos Inconsistentes: Si tu bitácora dice "Del 1 al 15", no puedes reportar una actividad del día 16.
  • Sin Evidencias: Siempre guarda pantallazos, correos o fotos que respalden lo que escribes.
Selección Alternativa (F-165)

Usa este formato para declarar tu modalidad de etapa productiva o solicitar un cambio de la misma.

  • Datos Básicos: Llena tu información personal y del programa de formación.
  • Selección: Marca con una X la alternativa elegida (Contrato de aprendizaje, Vínculo laboral, etc.).
  • Firmas: Debe estar firmado por ti y entregado a tu coordinador académico para el aval.
